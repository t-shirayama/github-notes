# PRテンプレート

PRテンプレートは、レビューに必要な情報を毎回そろえるための型です。最初から大きくしすぎず、よく漏れる項目だけを残します。

## 最小テンプレート

```md
## 概要

- 

## 確認したこと

- [ ] 正常系を確認した
- [ ] 異常系を確認した
- [ ] 影響範囲を確認した

## レビューしてほしい点

- 
```

## 少し詳しいテンプレート

```md
## 概要

- 

## 変更内容

- 

## 確認したこと

- [ ] テストを実行した
- [ ] 画面や動作を確認した
- [ ] 既存機能への影響を確認した

## レビューしてほしい点

- 

## 関連Issue

- 
```

## 使い方

- 大きくしすぎない。
- 毎回確認する項目だけ残す。
- 漏れが多い項目は後から追加する。
- PRの種類によって不要な項目は消してよい。
- 「確認していないこと」も必要なら明記する。

## 置き場所

リポジトリで共通テンプレートとして使う場合は、次の場所に置きます。

```text
.github/
└── pull_request_template.md
```

複数テンプレートを使い分ける場合は、次のように置けます。

```text
.github/
└── PULL_REQUEST_TEMPLATE/
    ├── feature.md
    ├── bugfix.md
    └── docs.md
```
